AI 圈热点周报

周期：2025年9月22日 - 9月28日

1DeepSeek V3.1-Terminus 发布，提升Agent能力

大模型 AI Agent 模型优化 DeepSeek 于9月22日发布了 V3.1-Terminus 版本更新。此次更新主要修复了先前版本中存在的中文和英文输出混杂的Bug，并对模型的 Agent 相关能力进行了专项提升，使其在执行复杂任务时更稳定可靠。

对普通人的价值分析

提效： 对于使用AI Agent执行自动化任务的用户来说，这意味着Agent的稳定性和理解力更强，能够减少因语言混乱或指令理解错误导致的失败，让自动化流程更顺畅。
赚钱： 基于更强大的Agent底层模型，开发者可以构建和销售更复杂的自动化工作流。例如，创建一个能自动完成市场调研、数据整理和报告撰写的“分析师Agent”，并将其作为SaaS服务提供给企业。

2腾讯开源混元图片3.0，精准生成汉字

AI绘画 开源模型 腾讯 腾讯宣布开源其最新的混元图片3.0模型。该模型在文生图领域取得显著突破，尤其擅长精准生成包含汉字的图像，解决了此前AI绘画“写不好中国字”的痛点，并已上线供用户在线体验。

对普通人的价值分析

提效： 设计师和内容创作者在制作海报、广告图或社交媒体图片时，可以直接生成包含准确中文字样的设计，无需再进行二次P图，大大提高了设计效率。
赚钱： 催生了新的商业设计服务。可以提供“AI辅助的国潮风海报”或“个性化带字头像/表情包”的快速设计服务。对于商家，则可以快速生成符合中国市场需求的营销物料。

3阿里发布全模态 Qwen3-Omni 模型，统一多模态输入

多模态 大模型 阿里 阿里巴巴发布了参数量为30B的Qwen3-Omni模型，这是一个原生的全模态（omni-modal）大模型。它能够在一个统一的架构中处理和理解文本、图像、音频和视频等多种模态的输入，实现了真正的“万能”输入。

对普通人的价值分析

提效： 用户与AI的交互将不再局限于文字。你可以直接扔给AI一段包含语音和画面的视频，让它进行总结或问答，AI能像人一样全面理解信息，极大提升了信息处理的广度和效率。
赚钱： 可以开发“多模态内容审查与标注”服务，帮助内容平台审核视频、音频的合规性。也可以为教育行业开发能够理解学生图文声并茂提问的“AI全科家教”应用。

4阿里发布WAN2.5模型，可生成带声音的视频

AI视频 文生视频 AIGC 继全模态理解之后，阿里巴巴在视频生成领域也发布了WAN2.5模型。该模型的一大亮点是实现了音视频同步生成，即在生成视频画面的同时，能够匹配相应的声音或配乐。目前，该功能已在通义万相App上线供用户体验。

对普通人的价值分析

提效： 视频创作者生成AI视频后，无需再耗费时间去寻找和匹配音效或背景音乐。真正实现“文生有声视频”，让AI视频创作流程缩短了一大步，内容也更具沉浸感。
赚钱： 可以为短视频博主、广告商提供“快速AI有声短片”制作服务。也适合用来批量生产带解说和BGM的“视频版”小说、新闻或教程，开辟新的内容变现渠道。

5快手发布可灵2.5 Turbo，更快更便宜

AI视频 降本增效 快手 快手发布了其可灵大模型的2.5 Turbo版本。新版本在保持高质量视频生成效果的同时，大幅提升了生成速度，并显著降低了使用成本，生成5秒1080p视频的价格下调了近30%。

对普通人的价值分析

提效： AI视频生成的门槛（时间和金钱）被进一步拉低。普通用户可以更快地将自己的想法变为视频，减少了等待时间，更低的成本也鼓励了更多人去尝试和创作。
赚钱： 成本的降低意味着利润空间的提升。从事AI视频制作服务的创作者可以用更低的成本交付作品，从而在市场上更具价格竞争力。也可以通过“走量”的方式，为客户提供低成本的批量视频素材生成服务。

6ChatGPT 上线 Pulse 功能，深夜为你整理知识卡片

ChatGPT 知识管理 AI助手 ChatGPT 为 Pro 用户上线了一项名为“Pulse”的新功能。该功能会基于用户与ChatGPT的对话历史和记忆，在用户休息的夜间自动进行思考和整理，并生成易于回顾的知识卡片，帮助用户巩固和发现知识。

对普通人的价值分析

提效： AI从一个被动回答的工具，变成了一个能主动为你“复习”和“总结”的学习伙伴。这有助于用户将碎片化的聊天内容系统化，更好地吸收和利用与AI交流获得的知识。
赚钱： 可以成为“AI知识管理顾问”，帮助用户（特别是学者、研究人员）建立高效的ChatGPT使用SOP，最大化利用Pulse功能构建个人知识库，并提供咨询服务。

7苹果全家桶支持MCP协议，第三方AI可操作App

Apple AI Agent 生态系统 苹果在新版的iOS和macOS中正式加入了对MCP（模型上下文协议）的支持。这意味着第三方AI模型（如ChatGPT）理论上将能够像Siri一样，直接调用和操作iPhone、iPad和Mac上的各种App，实现跨应用的复杂任务。

对普通人的价值分析

提效： 手机和电脑的操作方式将被颠覆。用户可以用一句话让AI完成过去需要打开多个App才能完成的复杂指令，例如“把我和朋友上周在相册里的合照，用VSCO P一下，然后发到Instagram上”。
赚钱： “快捷指令”的时代过去了，“AI指令”的时代到来了。精通MCP协议和AI调用的开发者可以设计和销售“超级AI指令”，解决特定场景下的复杂需求，在指令市场中获利。

8Kimi 上线 OK Computer 功能，内置云端虚拟机

Kimi AI Agent 代码执行 Kimi 上线了名为“OK Computer”的新功能，这是一个强大的 Agent 工具。它为AI提供了一个内置在云端的虚拟电脑环境，使其不仅能上网、读写文件，还能像程序员一样执行代码、安装软件，完成更复杂的数字化任务。

对普通人的价值分析

提效： AI的能力从“文科生”进化成了“理工科全能选手”。你可以让它帮你完成需要编程或使用特定软件才能完成的任务，例如“帮我写个Python脚本爬取这个网站的数据，并整理成Excel表格发给我”。
赚钱： 可以提供“AI自动化解决方案”服务，利用Kimi Agent为没有技术背景的客户解决数字化难题。例如，帮助小商家自动监控库存并生成采购订单，或为内容创作者自动完成视频转录和字幕制作。

9Kimi 推出会员功能，最低49元/月

Kimi 商业化 AI订阅 国产长文本大模型Kimi正式推出了会员订阅服务。其定价为每月49元起，付费会员可以享受更快的响应速度、更高的使用优先级以及更多的每日使用额度。作为限时优惠，Kimi还会赠送与订阅费等额的API调用量。

对普通人的价值分析

提效： 对于重度依赖Kimi进行长文阅读和分析的用户来说，付费可以解决高峰时段的排队和卡顿问题，确保AI工具的稳定性和高可用性，让工作学习不中断。
赚钱： 赠送的API额度是关键。这意味着你可以利用这些免费额度，开发一些轻量级的、基于Kimi长文本能力的应用或服务（如“简历分析与优化”、“论文摘要生成器”），进行商业化尝试，实现零成本创业。

10Github Copilot CLI 正式发布，AI进驻命令行

编程开发 CLI Github 在众多第三方工具之后，GitHub官方终于正式发布了自家的Copilot CLI（命令行界面）工具，并已开放公共预览。这使得开发者可以直接在终端（命令行）中，通过自然语言与Copilot进行交互，让AI执行代码构建、调试、版本管理等任务。

对普通人的价值分析

提效： 开发者，特别是后端和运维工程师，可以在不离开“黑框框”的情况下，完成大部分开发任务。忘记繁琐的命令和脚本，直接用自然语言指挥AI干活，开发体验更流畅。
赚钱： 可以录制和销售“AI时代的命令行实战”教程，帮助其他开发者提升效率。也可以为企业提供基于Copilot CLI的“自动化运维脚本”定制开发服务。

11阶跃星辰发布桌面Agent，可操作本地电脑

AI Agent 桌面自动化 RPA 国内AI创业公司阶跃星辰（StepFunction）发布了一款桌面Agent应用。与网页或云端Agent不同，它能直接操作用户的本地电脑，自带浏览器和终端，可以完成打开应用、整理文件、收发邮件等真实的本地事务。

对普通人的价值分析

提效： AI的自动化能力从云端延伸到了你的电脑桌面，成为真正的“电脑管家”。你可以让它帮你完成一系列跨应用的本地操作，例如“把我‘下载’文件夹里所有pdf文件的名字改成‘报告-日期’格式，然后移动到D盘的‘工作’文件夹里”。
赚钱： 这是RPA（机器人流程自动化）的平民化革命。普通人可以成为“个人自动化流程设计师”，为他人设计和配置解决日常电脑操作烦恼的自动化脚本，并进行销售。

12豆包内测AI输入法，支持智能语音输入

输入法 AI应用 豆包 字节跳动旗下的豆包App正在内测AI输入法功能。据悉，这款输入法将深度集成AI能力，除了更智能的联想和纠错外，还特别强化了语音输入功能，旨在提供更自然、更高效的移动端输入体验。

对普通人的价值分析

提效： 输入法是手机最高频的应用之一。AI的加持意味着打字和语音输入的效率和准确性会更高，AI还能根据上下文帮你润色、续写或翻译，让手机上的文字沟通变得更轻松。
赚钱： 对于内容创作者或社群运营者，可以利用AI输入法的润色和续写功能，快速生成高质量的文案或回复。这其中也可能催生出“AI输入法提示词工程师”这样的新角色，专门研究如何更好地利用输入法进行创作。

13Dify 1.9.0更新，可用工作流构建知识库

AI应用开发 知识库 Dify 低代码AI应用开发平台 Dify 更新至1.9.0版本。此次更新的核心是引入了“知识管道”（Knowledge Pipeline）功能，允许开发者通过可视化的工作流（workflow）方式，来构建和处理知识库，对文档的清洗、切片和索引过程进行精细化控制。

对普通人的价值分析

提效： 对于需要构建AI知识库（如AI客服、智能文档）的开发者或企业来说，这意味着可以更灵活、更精准地控制AI“学习”资料的方式，从而让最终的问答效果更好，减少AI“胡说八道”的概率。
赚钱： “AI知识库优化师”将成为一个有价值的岗位。你可以利用Dify的新功能，为企业提供专业的知识库构建和优化服务，帮助他们打造出回答更精准、效果更好的企业级AI应用。
