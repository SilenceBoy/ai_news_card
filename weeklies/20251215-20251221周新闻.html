AI 圈热点周报

周期：2025年12月15日 - 12月21日

1ChatGPT 正式开放“App Store”

OpenAI 生态变革 超级入口 ChatGPT 的“App Store”正式上线。开发者可以将自己的应用封装后提交，ChatGPT 页面新增了「商店」标签。这意味着用户在与 ChatGPT 聊天时，可以按需直接调用第三方应用（如订票、点餐、绘图），实现了从“对话”到“服务”的闭环。

对普通人的价值分析

提效： 手机里的 APP 可能会越来越少。未来你不需要打开“大众点评”找餐厅，再打开“滴滴”打车。直接在 ChatGPT 里说“帮我订一家附近的意大利餐厅并叫车过去”，它自动调用相关应用完成，操作链路被极致压缩。
赚钱： 全新的“GPT Store ASO”机会。 就像当年苹果 App Store 诞生一样，早期入驻的开发者将获得巨大的流量红利。现在去开发一个垂直领域的 AI 应用（如“AI 算命”、“AI 简历优化”）并上架，是目前最确定的流量风口。

2微软牵手 Kimi，国内 Office Copilot 有救了

微软 Kimi B端市场 微软宣布将与月之暗面（Kimi）进行 Agent 级合作。Kimi 将作为国内 Office 产品的 Copilot 驱动模型。这解决了微软 Copilot 因网络和合规问题在国内无法即插即用的痛点，实现了本土化的强强联合。

对普通人的价值分析

提效： 真正的“完全体”Office 来了。国内用户终于可以在 Word 里直接让 AI 续写长文、在 Excel 里一键分析数据、在 PPT 里自动生成页面，而且用的是懂中文语境的 Kimi，办公效率将有质的飞跃。
赚钱： “Office AI 培训与咨询”。由于 Kimi 的介入，国内数亿 Office 用户将面临工作流的升级。针对企业的“如何用 Kimi 版 Copilot 提升办公效率”的培训课程和内训服务将成为刚需。

3Google 发布 Gemini 3 Flash：速度提升 3 倍

大模型 Google 降本增效 Google 发布了 Gemini 3 Flash 模型。官方宣称其智能水平与 Gemini 3 Pro 几乎相当，但生成速度提升了 3 倍左右。这标志着“高智商”模型开始走向“实时化”和“低延迟”。

对普通人的价值分析

提效： “等待转圈”的时间大幅减少。对于需要大量生成内容（如翻译长文档、批量写文案）的用户，Flash 模型能在保证质量的前提下，让你的等待时间从 10 秒变成 3 秒。
赚钱： “实时交互应用开发”。3 倍的速度提升意味着它可以用于对延迟极度敏感的场景，比如开发“AI 实时同声传译助手”或“AI 游戏陪玩”，提供丝滑的对话体验。

4GPT-5.2-Codex 发布，OpenAI 强化编程能力

AI编程 OpenAI CodeX OpenAI 发布了基于 GPT-5.2 定向优化的 Codex 代码模型。同时，CodeX 产品现在也支持了 Skills 功能（与 Claude Code 对齐），进一步强化了其在复杂工程项目中的实战能力。

对普通人的价值分析

提效： 程序员的“外挂”升级了。新的 Codex 模型对最新技术栈的理解更深，生成的代码 Bug 更少。Skills 功能则允许你自定义常用的代码片段或工具，让 AI 更顺手。
赚钱： “AI 辅助开发外包”。利用 GPT-5.2-Codex 的强大能力，你可以一个人活成一支队伍，承接各种软件开发外包订单（如小程序、网站、脚本），大幅降低开发时间和人力成本。

5字节豆包发力：系统级 Agent 与多模态突破

字节跳动 AI手机 多模态 字节跳动动作频频：发布了 Doubao Seed 1.8（强化 Tool Use 和 OS Agent 能力），并正与 vivo、联想合作在手机系统层面预装 AI。同时发布了 Doubao-Seedance-1.5-pro 音画同生视频模型，以及将豆包助手封装为 Toa API 供开发者调用。

对普通人的价值分析

提效： 手机将从“智能手机”变成“智慧手机”。未来用 vivo 或联想手机，你不需要手动去设置里找开关，直接对手机说“帮我清理垃圾文件并把屏幕调暗”，系统级 Agent 会自动帮你完成复杂操作。
赚钱： “音画同生内容创作”。利用 Seedance-1.5-pro 模型，可以制作极具节奏感和视觉冲击力的音乐短视频（MV），在抖音等平台获取流量，且音画同步的特性让其质量远超普通 AI 视频。

6xAI Grok 与 Google functionGemma：语音 Agent 爆发

语音AI Agent 端侧模型 xAI 正式发布 Grok Voice Agent API，支持实时联网搜索和工具调用。Google 也发布了 functionGemma，这是一个可以在手机上运行的、支持工具调用的语音模型。两大巨头都在推动语音模型从“闲聊”走向“办事”。

对普通人的价值分析

提效： 语音助手不再是“人工智障”。你可以用语音让它们查天气、订闹钟、甚至控制家里的智能设备，而且因为支持工具调用，它们能真正解决复杂问题。
赚钱： “垂直领域语音机器人”。利用 Grok 的 API，可以开发特定行业的语音助手，比如“房地产电话销售机器人”或“酒店前台预订语音助手”，全天候自动处理客户需求。

7小米 MiMo-V2-Flash 免费不限量 API

免费资源 小米 API 小米发布了 MiMo-V2-Flash 模型，并宣布直到本月底前，其 API 不限量、不限速免费开放。这对开发者和想要尝鲜 AI 的用户来说是一波巨大的福利。

对普通人的价值分析

提效： 零成本试错。如果你想测试一些 AI 想法（比如写个自动摘要脚本），现在可以用小米的 API 随便跑，不用担心扣费，快速验证你的创意。
赚钱： “数据清洗与处理”。如果你手头有大量需要处理的数据（如给几万条评论打标签），趁着免费期，利用小米的 API 进行大规模的数据清洗和结构化处理，省下原本需要支付给其他厂商的昂贵 Token 费用。

8秘塔搜索新增 PPT 生成功能

办公AI 秘塔 PPT 秘塔搜索上线了类似 Google NotebookLM 的功能：用户可以将搜索结果报告直接生成为幻灯片（PPT），或者上传一个文档，让 AI 帮你加工成演示文稿。

对普通人的价值分析

提效： “搜完即讲”。当你做竞品分析或行业调研时，搜集完资料直接点一下生成 PPT，几分钟内就能拿着初稿去开会，省去了大量复制粘贴和排版的时间。
赚钱： “资料整理与代做 PPT”。可以利用秘塔的搜索+生成能力，快速承接大学生的作业资料查找或职场人的基础汇报 PPT 制作，以极高的效率赚取服务费。

9Google Opal 集成进 Gemini Gem，打造复杂流

工作流 Google 低代码 Google 将其工作流产品 Opal 集成进了 Gemini Gem。这意味着用户可以在 Gem 里开发逻辑更复杂的应用，不再局限于简单的提示词对话，而是可以构建包含判断、循环等逻辑的“小程序”。

对普通人的价值分析

提效： 个人工作流的自动化。你可以给自己做一个“Gem”，设定逻辑为：每天早上读取我的邮件 -> 提取关键信息 -> 写入日历 -> 发送到手机。一次配置，永久自动运行。
赚钱： “企业级 Gemini 应用定制”。企业需要处理复杂的业务流程（如报销审批、客户分流）。你可以利用 Opal 的能力，为企业定制开发高逻辑复杂度的 Gemini 内部应用，赚取技术开发费。

10Coze 首页变身 AI 编程，千问融入阿里系

低代码 生态融合 平台战略 字节的 Coze 平台更新了主页入口，重点转向“AI 编程”，支持直接开发智能体和工作流。同时，阿里的通义千问开始深度融入阿里系产品，首站是高德地图，预计后续将像元宝融入腾讯系一样，实现全生态覆盖。

对普通人的价值分析

提效： 地图导航变聪明了。在高德里用通义千问，你可以问“帮我规划一条不堵车、风景好、沿途有加油站的路线”，AI 能理解这种复杂的模糊需求。
赚钱： “Coze 编程应用开发”。Coze 转向编程意味着其能力上限被打开。你可以利用 Coze 开发更硬核的、具有商业价值的 SaaS 微应用（如跨境电商选品工具），并通过订阅收费。
